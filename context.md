# Project Context

This project will use the following stack:

- React (stable)
- Next.js
- Tailwind CSS
- MongoDB

## Notes

- React will be used for building the UI.
- Next.js will be used as the application framework.
- Tailwind CSS will be used for styling.
- MongoDB will be used as the database.

## UI Approach

- The frontend should follow a component-first approach.
- Reusable UI sections should be created as separate components and then composed inside the main `src/app/page.tsx` file.
- The home dashboard should stay simple in the beginning, with presentational components first and feature logic added gradually.
- This approach will make it easier to scale the app according to the architecture across acquisition, contracts, signing, and expense modules.
